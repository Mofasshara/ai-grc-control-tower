# Environment & Naming Convention

## Environment
DEV

## Naming Prefix
ai-grc

## Examples
Resource Group: rg-ai-grc-dev
Storage Account: stai-grc-dev
App Service: app-ai-grc-dev
PostgreSQL: pg-ai-grc-dev
Key Vault: kv-ai-grc-dev

## Purpose
Consistent naming ensures:
- Easy identification of resources
- Clear separation between environments
- Audit-friendly structure
- Predictable naming for automation
